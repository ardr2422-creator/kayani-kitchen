# Cyan Food — Site Vitrine

## Contexte du projet
Site vitrine pour Cyan Food, un fast-food français atypique proposant
une offre diversifiée (salades, bowls, burgers, snacks) présent en
Île-de-France. Démo commerciale destinée à convaincre le propriétaire
que ce site lui est indispensable pour attirer plus de clients et
diversifier ses canaux de vente (budget cible : 500€).

## Objectif du site
Ne pas vendre le site — convaincre le propriétaire qu'il en a besoin.
Le site doit avoir une âme, donner envie, inspirer confiance.
PAS minimaliste, PAS corporate : vivant, coloré, appétissant.
Se baser sur le site existant dans le dossier fourni pour la structure
des pages, mais changer complètement la direction artistique.

## Pages à créer (4 pages)
1. **index.html** — Accueil
   - Hero section (titre, accroche, chiffres clés)
   - Section "Notre histoire"
   - Section avis clients (authentiques, sans nombre total)
   - Footer

2. **menu.html** — Menu complet
   - Burgers, salades, bowls, snacks, boissons
   - Construire à partir des fichiers du sous-dossier fourni
   - Prix réalistes pour un fast-food français

3. **contact.html** — Contact
   - Formulaire QCM interactif
   - Infos pratiques

4. **nous-trouver.html** — Localisation
   - Adresse, horaires, carte

## Identité visuelle
- Couleurs, moodboard et images : tout est dans le sous-dossier fourni
- Respecter strictement le moodboard pour les couleurs
- Style : avec une âme, coloré, street food premium — PAS minimaliste
- Typographie forte et identitaire (Google Fonts)
- Placeholders CSS uniquement quand aucune image disponible
